# mazebot

to run:

install libpng package

to compile:
```
gcc -o maze maze.c -lpng
```
python3 mazebot.py starts a discord bot that accepts dash commands

slashbot accepts slash commands (is a user application)

keep tokens in ~/bot_tokens
